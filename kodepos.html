Api Kodepos Indonesia¶

Kalarau.net menyediakan listing data untuk kodepos yang ada di Indonesia, mungkin belum lengkap maka dari itu jika anda berkenan untuk berkontribusi, saya menyediakan endpoint khusus untuk menambahkan data tersebut.

Catatan

Seluruh Endpoint yang didefinisikan pada laman ini mengacu pada base-url http://kalarau.net/api/v1/

Detail Kodepos¶

GET /kodepos/(int: kodepos)¶

Memberikan detail informasi terhadap kodepos yang diberikan

Contoh Request

GET /kodepos/17145
Host: kalarau.net

Contoh response

{
    "kodepos": [
        {
            "_id": 1285,
            "kodepos": "17145",
            "provinsi": "Jawa Barat",
            "kabupaten": null,
            "kota": "Bekasi",
            "kecamatan": "Bekasi Barat",
            "kelurahan": "Jaka Sampurna"
        }
    ]
}

Daftar Kodepos¶

Berikut adalah daftar endpoint untuk melihat daftar kodepos, diurutkan berdasarkan tingkat Pemerintahan daerah.

Pencarian Kodepos¶

GET /kodepos/search¶

Memberikan daftar kodepos yang mirip dengan kata kunci pencarian.

Query Parameters:
 
  • q (string) – kata kunci pencarian

Contoh Request

GET /kodepos/search?q=bekasi
Host: kalarau.net

Contoh Response
